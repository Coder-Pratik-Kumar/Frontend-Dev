Fluid Layouts

Use relative units like percentages or vw/vh to make layouts adapt to screen sizes.

width: 80%; max-width: 1200px;

Flexible Media

Ensure images and videos resize properly within containers.

img { max-width: 100%; height: auto; }

Breakpoints

Use media queries to adjust layouts and fonts at specific screen widths.

@media (max-width: 600px) { body { font-size: 14px; } }
